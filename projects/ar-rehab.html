AR Pediatric Rehabilitation Games Suite

Ongoing Project

Overview

An augmented reality–based games suite designed to support pediatric rehabilitation by encouraging children to perform therapeutic exercises through interactive AR gameplay.

Problem Statement

Traditional rehabilitation exercises for children can be repetitive and disengaging, leading to reduced participation and slower recovery. There is a need for an interactive and motivating rehabilitation approach tailored for pediatric patients.

Solution

The system uses augmented reality and computer vision to convert rehabilitation movements into interactive games. Children perform prescribed physical actions while receiving real-time visual feedback, making therapy engaging and goal-oriented.

Technologies Used

  • Augmented Reality frameworks
  • Computer Vision
  • Python
  • Game Logic & Motion Tracking

Current Work

Improving motion detection accuracy, refining AR interactions, and designing child-friendly game mechanics aligned with therapy goals.

Planned Enhancements

Adding progress tracking, personalised rehabilitation routines, and remote monitoring features for therapists.
