AR Pediatric Rehabilitation Games Suite

Ongoing Project

Overview

AR Pediatric Rehabilitation Games Suite is an interactive motion-based therapy platform designed to assist pediatric rehabilitation through engaging mini-games. Using computer vision and gesture tracking, therapeutic movements are transformed into simple games that encourage children to perform physical exercises consistently while keeping them motivated and entertained.

Problem Statement

Rehabilitation exercises for children are often repetitive and can quickly become boring, which reduces participation and affects recovery progress. Therapists require an engaging system that motivates children while still capturing meaningful movement data during therapy sessions.

Solution

This project introduces a computer-vision driven rehabilitation game system where children interact with on-screen targets using hand movements. Real-time motion tracking converts therapy exercises into gameplay tasks such as popping balloons, allowing rehabilitation to feel more like a game while maintaining therapeutic value.

Current Features

  • Balloon-popping therapy game using hand gesture interaction
  • Dynamic balloon spawn logic with configurable spawn timing
  • Target spawn interval tuning (currently 3.5 seconds)
  • Velocity-based weighting to analyse motion speed
  • Logarithmic scaling for improved motion sensitivity
  • Interaction heatmap generation to visualise movement patterns

Technologies Used

  • Python
  • OpenCV
  • MediaPipe (hand tracking)
  • Computer Vision
  • Game Logic & Motion Analysis

Current Work

The project is currently focused on refining motion detection accuracy, improving gesture responsiveness, and analysing player interaction data through heatmaps to better understand rehabilitation movement patterns.

Planned Enhancements

  • Additional rehabilitation mini-games
  • Adaptive difficulty based on patient movement behaviour
  • Progress tracking and performance analytics
  • Therapist dashboard for monitoring rehabilitation sessions
