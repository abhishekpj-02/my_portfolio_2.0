Gesture-Based Authentication

Overview

This project implements a gesture-based authentication system where a user is verified based on unique hand gestures captured using computer vision techniques.

Problem Statement

Traditional authentication methods like passwords and PINs are prone to security risks. This project explores gesture recognition as an alternative, secure, and intuitive authentication method.

Solution

A machine learning model is trained to recognize predefined hand gestures using OpenCV and TensorFlow. A Flask API handles verification, and a Tkinter GUI provides user interaction.

Technologies Used

  • Python
  • TensorFlow
  • OpenCV
  • Flask
  • Tkinter

Future Improvements

Improve gesture robustness, add multi-user profiles, and integrate the system with edge devices.
